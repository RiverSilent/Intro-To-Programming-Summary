Summary of L1

The web

The web consistes of four major components.

  • Client(Browsers)
  • The Internet(Wireless network)
  • HTTP (The protocol using which the client communicates with the server).
  • Servers (Computers that serve the file requested by the clients).

HTML

HTML consists of contents within tags.First there is an opening tag which is then followed by the content and then there is the closing tag.

There are two types of elements - inline and block.

Block elements form an imaginary box around them.Inline elements just wrap the subsequent content next to them.

It's OK to forget the tags!

Whitespace

HTML converts any number of whitespaces between contents to a single whitespace.

There are two ways to put contents in separate lines in HTML.

The first is to use br tag. The next one is to use the p tag.

Summary of L2

The four stages of learning

Theere are four stages in learning to code(actually, anything):

  • Ignorance
  • Awareness
  • Ability
  • Fluency

Developer Tools

The developer tools available in the browser are an excellent way to experiment making changes in the site.There is a separate styles section that is used to change the look of the elements displayed.

HTML-CSS-DOM

HTML describes the structure of the webpage, ie how the elements will be laid out.

CSS describes the visual styling of the webpage.

The browser converts the HTML to a tree like structure,known as the Document Object Model.

Boxes

Every element in the webpage is represented by a box.

Looking at the design mock of the web page, we first decide all the boxes that will be enclosing the elements.

CSS

CSS stands for Cascading Style Sheets.It means that the most specific style rule is always applied to an element.

Don't Repeat Yourself!

An important part in programming is the concept of Don't Repeat Yourself.To achieve this, we can have classes assigned to different HTML elements. Then all we need to do is to assign a particular style to that class once and all the elements of that class will have that style.

Elements can also have an ID assigned to them.An element can have only one ID and there can only be one element with the same ID in the webpage.However, an element can have multiple classes and the same class name can be assigned to multiple elements.

Box Model

Each element is enclosed by a box.The box has four main parts-

  • Content - The main content
  • Padding - The space around the content
  • Border - The border around the element
  • Margin - The space outside the element.

If two or more elements are to be positioned side by side,we set the display property of their parent div to be flex.

Code,Test,Refine

This cycle of code,test,refine is iterated upon while coding.The dev tools ar a great way to explore the possibilities before making the final change in the HTML or CSS file.

Verifying CSS and HTML

HTML and CSS can be verified using HTML and CSS validators.
